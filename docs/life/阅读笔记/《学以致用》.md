# 《学以致用》

“应用”是学习的目标，“会用”是学习成效的度量。

想要所学会用，最好的方式是即学即用。

“用”的方式很多，如：

* 参与讨论
* 实践
* 教授

其核心在于人的“主动思考/创作”。“被动接受”是非常低效和易忘的。

![阅读金字塔](../../asset/78b2fa40-bc90-11eb-8897-81db1c78e0a0.jpg)

难以实践的知识，如理论、概念等，快速浏览即可，起到“了解/知道”的效果。未来在其相关任务的实践中，会自然形成记忆/习惯。

浅层学习注重输入，深层学习注重输出。

1. 获取高质量知识；
2. 深度缝合新知识；
3. 输出成果去教授。
